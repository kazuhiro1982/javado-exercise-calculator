# コマンドライン電卓アプリ

## 概要

コマンドラインで四則演算ができるアプリです。

## 使い方

1. このリポジトリをクローンします。
2. クローンしたディレクトリに移動します。
3. ビルドを実行します。
    ```bash
    $ ./gradlew jar
    ```
4. アプリケーションを実行します。
    ```bash
    $ java -jar app/build/libs/app.jar "1 + 2"
    ```
5. 結果が表示されます。
    ```bash
    result: 3
    ```

## テスト

このコードには恐らくバグや期待しない動作がありそうです。

`app`フォルダ配下の `src/test/java` にいくつか簡単なテストコードがあります。
